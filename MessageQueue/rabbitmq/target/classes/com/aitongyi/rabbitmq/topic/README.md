# Topic Exchange


生产者----交换机-----队列-----消费者

- 1，一个生产者，多个消费者
- 2，每个消费者都有自己的队列
- 3，生产者没有将消息直接发送给队列，而是发送给交换机
- 4，每一个队列都要先绑定到交换机上
- 5，生产者将消息发送到交换机上，经过交换机，到达队列，就能够实现一个消息被多个消费者消费。


将路由键和某模式进行匹配，此时，队列需要绑定要一个模式上
符号“#”匹配一个，或者多个词 
符号“*”匹配不多不少一个词
因此，“audit.#”能够匹配到“audit.irs.corporate” 
但是“audit.*” 只会匹配到“audit.irs”
